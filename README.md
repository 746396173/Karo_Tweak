### 安装

**1. 安装**

* 点击`clone or download`按钮下载 QQPlugin 至`下载`目录并解压，打开Termimal，执行 `./Other/Install.sh`即可。


**2. 安装完成**

* 登录微信，看到**QQ小助手**即安装成功。

---

### 卸载

在`Terminal`(终端)打开该项目，运行 `./Other/Uninstall.sh` 即可.
